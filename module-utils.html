Module: utils

Provides utility functions and basic protection mechanisms for the application. This module includes helper functions for escaping strings, debouncing user input, and handling redirection with error messages. It also adds client-side protection by preventing right-click, blocking developer tool shortcuts, and detecting debugger usage.
