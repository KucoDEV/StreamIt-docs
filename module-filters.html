Module: filters

Manages the creation, population, and handling of filtering options for films and series. This module includes all logic required to populate dropdown filters (genres, years, directors, actors, creators), generate filter options from the dataset, and apply the selected filters to dynamically update the visible content. It also provides utility functions to reset filters and refresh the content grids accordingly.
