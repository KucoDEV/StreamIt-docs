Module: storage

Provides access to and manipulation of watch progress data stored in localStorage. This module defines global datasets (`filmsData`, `seriesData`) and manages user watch state for both films and series, including whether content has been watched and the last watched timestamp. Data is persisted across sessions using the `watchedContent` key in localStorage.
